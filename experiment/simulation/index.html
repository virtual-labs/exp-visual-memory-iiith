Instructions: How to Perform the n-Back Task
  • Set the value of "n" for the N-Back Test. This determines how many steps back you need to compare letters.
  • Click "Start" to begin the test.
  • Once the letters start appearing, you need to identify if the current letter matches the letter shown "n" steps earlier.
  • Click "Yes" if the letters match, and "No" if they don't.
  • Your score will increase every time you answer correctly. Keep track of your score as you progress.
